# HR 简历筛选台账

本程序用于 HR 本地浏览招聘/智能招聘系统时，把当前候选人页面采集、打分、沉淀为台账。

## 启动

双击：

```bat
启动HR简历台账.bat
```

然后浏览器会打开：

```text
http://127.0.0.1:8765
```

## 使用流程

1. 在本地控制台填写岗位关键词，例如：`Python, 数据分析, SQL, 本科, 3年`
2. 点击「启动招聘浏览器」
3. 在新打开的 Edge/Chrome 里登录智联、BOSS、猎聘、Moka、北森等系统
4. 打开候选人详情页
5. 回到控制台点击「刷新页面列表」→ 选择候选人页面 →「采集当前页」
6. 核对识别结果后点「保存入台账」
7. 在台账中维护状态并导出 CSV

## 数据位置

```text
data/hr_resume_ledger.sqlite3
```

默认只保存在本机，不保存账号密码。
